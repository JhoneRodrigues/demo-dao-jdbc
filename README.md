# demo-dao-jdbc

### Sobre o projeto

Este projeto tive como objetivo estudar e praticar JDBC, uma API comumente utilizada em Java para consultar e modificar banco de dados.

Mesmo com pouco conhecimento em linguagem SQL, desenvolvi este projeto focando no padr√£o DAO (Data acces object) contendo testes meramente demonstrativos diretamente no banco de dados SQL.
